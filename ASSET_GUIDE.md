## Tools for Creating Screenshots

### Option 1: Native Screenshot Tools
1. **macOS:** 
   - Use Cmd+Shift+4 for area selection
   - Use Cmd+Shift+3 for full screen
   - Resize in Preview app or online tool

2. **Chrome DevTools:**
   - Right-click extension popup → Inspect
   - Use Device Toolbar (Cmd+Shift+M)
   - Set custom size: 1280x800
   - Take screenshot

### Option 2: Browser Extensions
- **Full Page Screen Capture** (Chrome extension)
- **Awesome Screenshot** (Chrome extension)
- **Nimbus Screenshot** (Chrome extension)

### Option 3: Design Tools for Promo Tiles
- **Canva** (free, templates available)
- **Figma** (free, web-based)
- **GIMP** (free, open-source)
- **Photoshop** (paid)

## Step-by-Step Screenshot Process

### For Screenshot 1 (Hero Shot):
1. Load extension in Chrome (unpacked mode)
2. Click extension icon to open popup
3. Type "bil" in the search field
4. Take screenshot of the popup showing "en bil ✅"
5. Open screenshot in image editor
6. Resize to 1280x800px if needed
7. Add annotation text: "Check any Swedish word in under 1 second"
8. Save as PNG

### For Screenshot 2 (Context Menu):
1. Open any Swedish website (e.g., svenska.se)
2. Select a word like "hus"
3. Right-click to show context menu
4. Take screenshot showing the menu option
5. Resize to 1280x800px
6. Add annotation: "Right-click any word on any website"
7. Save as PNG

### For Screenshot 3 (Pattern Learning):
1. Open extension popup
2. Type a word that triggers pattern detection (e.g., "station")
3. Take screenshot showing pattern explanation
4. Resize to 1280x800px
5. Add annotation: "Understand the patterns behind en/ett"
6. Save as PNG

### For Screenshot 4 (Offline):
1. Disable internet connection (or use Chrome DevTools → Network → Offline)
2. Open extension popup
3. Perform a lookup
4. Take screenshot showing it still works
5. Resize to 1280x800px
6. Add annotation: "1000+ words available offline"
7. Save as PNG

### For Screenshot 5 (Design):
1. Open extension popup
2. Show multiple states (empty, searching, result, error)
3. Arrange them artfully in a single image
4. Resize to 1280x800px
5. Add annotation: "Beautiful, thoughtful design inspired by Sweden"
6. Save as PNG

## Image Optimization Tips

- **File Size:** Keep screenshots under 1MB each
- **Format:** PNG for screenshots with text, JPEG for photos
- **Quality:** Use high quality but compress if file is too large
- **Naming:** Use descriptive names like `screenshot-1-hero.png`
